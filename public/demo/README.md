# Public demos

- [`first-time-walkthrough.html`](./first-time-walkthrough.html) — interactive first-time user walkthrough
- [`first-time-walkthrough.mp4`](./first-time-walkthrough.mp4) / [`.webm`](./first-time-walkthrough.webm) — screen recording

Regenerate the recording from the repo root:

```bash
npm run record-walkthrough
cp demo/videos/first-time-walkthrough.* public/demo/
cp demo/first-time-walkthrough.html public/demo/
```
